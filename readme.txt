### MeroUsers - User Management System

### Overview

MeroUsers is a web-based user management system that allows users to register, log in, and manage their profiles. The system is built using PHP and MySQL, with a simple and user-friendly interface.

### Features

User authentication (Login, Registration, Logout)

Profile management (Update personal details and profile image)

Secure password handling with hashing

Responsive UI with Bootstrap


### Technologies Used

PHP (Backend)

HTML, CSS, Bootstrap (Frontend)

MySQL (Database)

JavaScript (For UI enhancements)



### Installation & Setup

Clone or download the project files.

Set up a local server using XAMPP, WAMP, or laragon or any PHP-supported environment.

Import the provided database (user_management.sql) into MySQL.

Configure database credentials in config.php.

Start the local server and access the application via http://localhost/website/.


### Usage

Register a new account or log in to an existing account.

Access and update your profile information.

Upload a profile image if desired.
